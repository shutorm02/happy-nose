# Happy Nose
### All are helpless in front of this nose.

- When: 疲れたときに
- Who: 動物がそばにいない人が
- Why: こころを落ち着かせるために
- What: 犬の鼻を
- How: 無心に押す

■ **鼻を押すと…**</br>
　まばたきする、</br>
　鼻をムズムズさせる、</br>
　耳が動く。</br>

■ **鼻を押さず、放置していると…**</br>
　鼻を近づけてくる、</br>
　首をかしげる、</br>
　まばたきする。</br>
